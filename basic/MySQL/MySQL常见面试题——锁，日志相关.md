# MySQL常见面试题——锁相关

## MySQL中有哪些锁？

- **全局锁**：让整个表变成只读状态
- **表级锁**：
	- **表锁**：所有线程均只读
	- **元数据锁**：防止用户执行CRUD操作时有线程修改表结构
	- **意向锁**：进行增改删前先加上`意向独占锁`，然后对该记录加锁。**用于快速判断该数据是否被上锁**
- **行级锁**：对行加锁
- **记录锁**：对记录加锁，锁分为S锁与X锁，满足读写互斥，读读互斥
- **间隙锁**：用于可重复读，解决幻读用
- **临键锁/Next-Key Lock**： Record Lock + Gap Lock 的组合，锁定一个范围，并且锁定记录本身
