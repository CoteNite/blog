## SpringBoot4

就在不久，SpringBoot4震撼首发，主要的几个更新点在于：

- 全面支持JSpecify，整个系统实现基于JSpecify的空安全
- 全面兼容Java25，与Java始终并肩而行

现在我们一起看看它的几个核心功能更新

## 1.API版本控制

现在我们可以基于API对接口进行版本控制

```kotlin
@RestController  
@RequestMapping("/api")  
class HelloWorldController {  
      
    @GetMapping("/hello", version = "v1", produces = [MediaType.TEXT_PLAIN_VALUE])  
    fun hello(): String {  
        return "Hello World!"  
    }  
      
    @GetMapping("/hello", version = "v2",produces = [MediaType.TEXT_PLAIN_VALUE])  
    fun hello2(): String {  
        return "Hello World2!"  
    }  
      
}
```
