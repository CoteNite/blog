# 对话循环

Claude Code的对话主循环是一个异步生成器，他类似一个可以随时暂停或者恢复的“协程”，在对话循环和UI渲染之间建立了一条实时的事件管道

```mermaid
flowchart TD
    %% 第一层
    subgraph layer1["🌊 第一层：流式响应 (Streaming)"]
        direction LR
        e1["事件 A"] -->|yield| e2["事件 B"] -->|yield| e3["事件 C"]
        note1(["<b>实时渲染</b><br/>降低首字节延迟"])
    end

    %% 第二层
    subgraph layer2["🛑 第二层：生命周期控制"]
        direction TB
        cancel["调用 .return()"] --> clean[["finally { 资源回收 }"]]
        clean --> scene2(["<b>场景</b>：用户中断操作"])
    end

    %% 第三层
    subgraph layer3["⚖️ 第三层：背压控制 (Backpressure)"]
        direction TB
        bp["按需拉取模式"] --> mem["内存水位监控"]
        mem --> scene3(["<b>场景</b>：海量数据处理"])
    end

    %% 建立垂直堆叠关系
    layer1 ~~~ layer2
    layer2 ~~~ layer3

    %% 样式修正
    classDef primary fill:#e1f5fe,stroke:#01579b,stroke-width:2px
    classDef secondary fill:#fff3e0,stroke:#ef6c00,stroke-width:2px
    classDef accent fill:#f1f8e9,stroke:#33691e,stroke-width:1px
    
    class e1,e2,e3,cancel,bp primary
    class clean,mem secondary
    class note1,scene2,scene3 accent
```
