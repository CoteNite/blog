## NLP（Nature Language Processing）/自然语言处理

NLP是一种理解，解释和生成人类语言的技术，是利用机器生成处理和生成人类语言的技术

常见的NLP任务有以下这些：

| **任务类型**      | **代表任务**        | **核心目标**                    |
| ------------- | --------------- | --------------------------- |
| **基础预处理**     | 中文分词、子词切分       | 将连续文本打破为计算机可处理的最小语义单位。      |
| **语言结构分析**    | 词性标注 (POS)      | 识别单词在句中的语法功能（名、动、形）。        |
| **信息提取 (IE)** | 实体识别 (NER)、关系抽取 | 从非结构化文本中提取出“谁”、“在哪”、“干了什么”。 |
| **文本理解与分类**   | 文本分类、情感分析       | 判定文本的属性、类别或情绪倾向。            |
| **内容生成与转化**   | 文本摘要、机器翻译、自动问答  | 实现语言间的转换或根据上下文创造新文本。        |
